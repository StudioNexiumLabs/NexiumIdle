## ⚡ Nexium Idle — by Nexium Labs

🪐 Idle Game 2D futuriste développé avec Godot

### 🎮 Vision du jeu

Nexium Idle est un idle game futuriste où le joueur génère de l’énergie, améliore ses systèmes et explore un univers mystérieux.  
Objectif : un MVP simple, fun et évolutif, parfait pour une première version jouable.

**Mécaniques principales :**
- ⚡ Production d’énergie  
- 🤖 Drone collecteur  
- 📈 Upgrades & améliorations  
- 🌌 Exploration  
- 🔁 Prestige / Reset stratégique  
- 🧪 Technologies  

**Tech :**
- Godot Engine (GDScript)  
- Pixel Art  
- Architecture simple & modulaire  
- Systèmes évolutifs pour itérations rapides  

---

## 🌐 Vision Web3 (future)

La version MVP du jeu **ne contient pas de Web3**.  
Cependant, Nexium Labs explore une vision long terme intégrant des fonctionnalités Web3 optionnelles :

- 🎴 Objets ou ressources uniques (NFT non spéculatifs)  
- 🔐 Progression persistante et sécurisée  
- 🪙 Économie contrôlée par les joueurs  
- 🌌 Systèmes inter‑jeux (interopérabilité)  

Ces éléments **ne font pas partie du développement actuel**, mais représentent une direction possible pour l’avenir du projet.
