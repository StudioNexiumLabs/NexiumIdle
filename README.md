# Nexium Idle — by Nexium Labs ⚡🤖🌌

Nexium Idle est un idle game 2D futuriste développé avec Godot.
Le but : créer un MVP simple, fun et évolutif.

## 🎮 Vision du jeu
- Production d’énergie
- Drone collecteur
- Upgrades
- Exploration
- Prestige

## 🧪 Technologies
- Godot Engine
- Pixel Art
- Systèmes simples et modulaires

## 🤝 Recrutement Godot
Nous recherchons un développeur Godot (débutant ou junior accepté).
Formulaire : [(https://forms.gle/WZLuUP4zvPE9ud8E6)]

## 📁 Structure du projet
/assets — sprites, icônes, logo  
/code — scripts Godot  
/design — flowcharts, documents  
/docs — notes, roadmap  

## 🚀 Objectif
Créer un MVP jouable et propre, puis itérer ensemble.
